## 0.1.0 - 2026-01-31

### 变更类型：新增

### 新增功能
- 多厂商接入与授权管理：支持 Cloudflare（Global API Key）与 DNSPod（Token）接入，凭据本地加密存储
- 域名列表聚合：实时拉取聚合列表，支持关键字搜索与按厂商筛选
- 解析记录 CRUD：支持 A/AAAA/CNAME/TXT/MX/NS/SRV/CAA 单条记录新增/修改/删除与冲突策略选择

### 修改功能
- 无

- 无
